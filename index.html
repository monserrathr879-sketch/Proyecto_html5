¿Qué es HTML5?

HTML5 es la última versión del lenguaje de marcado HTML, utilizado para crear y estructurar el contenido de las páginas web. Sirve para organizar elementos como textos, imágenes, videos, enlaces y formularios, permitiendo que los navegadores interpreten y muestren correctamente la información. Es importante porque facilita el desarrollo de sitios web modernos, ya que incluye nuevas etiquetas y funcionalidades que mejoran la accesibilidad, el rendimiento y la compatibilidad con distintos dispositivos. Gracias a HTML5, es posible crear páginas más interactivas, dinámicas y adaptables sin depender tanto de tecnologías externas.

Estructura básica de HTML5

<!DOCTYPE html> <html> <head> <title>Mi página</title> </head> <body> </body> </html>

La etiqueta <!DOCTYPE html> se utiliza para indicar al navegador que el documento está escrito en HTML5, lo que asegura que se interprete correctamente. La etiqueta <html> es la raíz del documento y contiene todo el contenido de la página web. La etiqueta <head> almacena información importante sobre la página, como el título, enlaces a hojas de estilo, metadatos y scripts, pero no es visible directamente en la página. La etiqueta <body> contiene todo el contenido visible para el usuario, como textos, imágenes, enlaces, botones y demás elementos que se muestran en el navegador.

Etiquetas principales

h1 - h6

Las etiquetas h1 a h6 se utilizan para definir títulos y subtítulos en una página web. h1 es el título más importante y grande, mientras que h6 es el menos importante. Sirven para organizar el contenido y mejorar la estructura del sitio.

p

La etiqueta p se utiliza para crear párrafos de texto. Permite organizar la información en bloques, facilitando la lectura y comprensión del contenido.

img

La etiqueta img se usa para insertar imágenes en una página web. Requiere atributos como "src" para indicar la ruta de la imagen y "alt" para describirla.

a

La etiqueta a se utiliza para crear enlaces o hipervínculos que permiten navegar entre páginas web o hacia diferentes secciones dentro del mismo sitio.

button

La etiqueta button se usa para crear botones interactivos que pueden ejecutar acciones, como enviar formularios o activar funciones con JavaScript.

ul / ol / li

Las etiquetas ul (lista desordenada), ol (lista ordenada) y li (elemento de lista) se utilizan para crear listas. Las listas ayudan a organizar información de manera clara y estructurada.

HTML5 semántico

header

La etiqueta header se utiliza para definir el encabezado de una página o sección. Generalmente contiene el título, logotipo o información introductoria del sitio web.

nav

La etiqueta nav se usa para crear una sección de navegación. Incluye enlaces que permiten al usuario moverse entre diferentes páginas o secciones del sitio web.

main

La etiqueta main representa el contenido principal de la página. Aquí se coloca la información más importante que el usuario debe ver.

section

La etiqueta section se utiliza para agrupar contenido relacionado dentro de la página. Ayuda a dividir el contenido en partes organizadas y fáciles de entender.

article

La etiqueta article se usa para contenido independiente que puede entenderse por sí solo, como noticias, publicaciones o artículos de blog.

footer

La etiqueta footer define el pie de página. Generalmente contiene información como derechos de autor, enlaces adicionales o datos de contacto.

Formularios en HTML5

Los formularios en HTML se utilizan para recopilar información del usuario, como datos personales, comentarios o respuestas. Son una parte fundamental en las páginas web, ya que permiten la interacción entre el usuario y el sitio. La etiqueta form es el contenedor principal del formulario y dentro de ella se colocan todos los elementos que capturan información. La etiqueta label se utiliza para describir el campo que el usuario debe llenar, facilitando la comprensión. La etiqueta input permite crear diferentes tipos de campos, como texto, correo, contraseña, entre otros. La etiqueta textarea se usa para ingresar textos largos, como comentarios o mensajes. La etiqueta select crea listas desplegables donde el usuario puede elegir una opción. La etiqueta button se utiliza para crear botones que envían el formulario o realizan alguna acción.

Multimedia en HTML5

HTML5 permite trabajar con contenido multimedia de forma sencilla sin necesidad de plugins externos. Gracias a sus nuevas etiquetas, es posible insertar imágenes, reproducir audio y mostrar videos directamente en la página web, mejorando la experiencia del usuario y la compatibilidad con distintos dispositivos.

Imágenes

La etiqueta img se utiliza para insertar imágenes en una página web. Sus atributos principales son src, que indica la ruta de la imagen, y alt, que proporciona una descripción alternativa.

Audio

La etiqueta audio permite reproducir archivos de sonido en la página web. Puede incluir controles como reproducir, pausar y ajustar volumen mediante el atributo controls.

Video

La etiqueta video se utiliza para mostrar videos directamente en la página. Permite agregar controles, definir el tamaño y utilizar múltiples formatos de video.

Accesibilidad

El atributo alt es importante porque describe las imágenes para personas con discapacidad visual y para cuando la imagen no carga, mejorando la accesibilidad y el SEO.

Conclusión

HTML5 es fundamental en el desarrollo web moderno, ya que facilita la creación de páginas más estructuradas, interactivas y accesibles. Gracias a sus nuevas etiquetas y funcionalidades, los desarrolladores pueden crear sitios web más eficientes, adaptables y compatibles con diferentes dispositivos sin depender de tecnologías externas.
